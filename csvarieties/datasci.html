Data Science

Data Science is a multidisciplinary field that involves extracting insights and knowledge from data through various techniques, algorithms, and processes. Some key concepts of this emphasis are data analysis, data manipulation, data visualization, machine learning, and statistical modeling.



Some objectives of this emphasis is:

- Teach the skills needed to be a data scientist.

- Be able to understand foundational statistical concepts.

- Learn how to gather and clean data.

- Discover how to explore data visually and statistically to uncover patterns, relationships, and anomalies.

- Explore techniques to create effective data visualizations that communicate insights clearly to both technical and non-technical audiences.

- Study machine learning algorithms and understand how to apply these algorithms to real-world problems

- Learn how to assess the performance of your machine learning models using metrics like accuracy, precision, recall, F1-score, and cross-validation.
